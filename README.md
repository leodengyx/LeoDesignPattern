LeoDesignPattern
================
LeoDesignPattern project is mainly trying to implement the famous design patterns in Python.

Creational Patterns
-------------------
* Abstract Factory
* Factory Method
* Builder
* Prototype
* Singleton

Structural Patterns
-------------------
* Adapter
* Bridge
* Composite
* Facade
* Proxy

Behavior Patterns
-----------------
* Chain of Responsibility
* Command
* Mediator
* Memento
* Observer
* State
* Strategy
* Visitor
